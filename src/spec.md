# Specification

## Summary
**Goal:** Add an animated gaming background to the login queue screen and social media links (YouTube and Instagram) to both the queue and store pages.

**Planned changes:**
- Add custom animated video background to the login queue screen that displays dynamic gameplay footage
- Add YouTube and Instagram icons at the bottom of the LoginQueue component
- Add YouTube and Instagram icons at the bottom of the storefront page
- Add Instagram URL configuration field to the admin panel's payment configuration section
- YouTube links to https://youtube.com/@fros7yyt?si=u4-j8LrTkNuj7wUo
- Instagram URL is configurable by admin through the payment configuration

**User-visible outcome:** Users will see an animated gaming background while waiting in the queue and can access YouTube and Instagram social links from both the queue and store pages. Admins can configure the Instagram URL through the admin panel.
